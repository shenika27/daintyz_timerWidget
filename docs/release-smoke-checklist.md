기능 또는 의존성을 변경해 새 APK를 배포하기 전, 실제 Android 기기에서 아래 항목을 확인한다.

## 필수 확인

### 1. 신규 설치와 위젯

- [ ] 기존 앱 데이터를 지우거나 앱을 제거한 뒤 새 APK를 설치한다.
- [ ] 홈 화면에 2x2 위젯을 추가한다.
- [ ] 기본 캐릭터와 타이머 스킨이 내장 스킨 `cha01`로 표시된다.
- [ ] `+`, `-`, 시작, 일시정지/재개, 중지 버튼과 시간 영역 탭이 의도한 상태 전환을 수행한다.

### 2. 알림 권한

Android 13 이상에서는 각각 한 번씩 확인한다.

- [ ] 알림 권한을 허용한 상태에서 진행·완료 알림이 정상 표시된다.
- [ ] 알림 권한을 거부한 상태에서도 타이머와 위젯이 정상 동작하고 앱이 종료되지 않는다.

### 3. 화면 OFF와 완료

- [ ] 1분 타이머를 시작하고 화면을 끈다.
- [ ] 완료 시점 이후 화면을 켜 위젯이 `COMPLETE` 상태와 `00:00`을 표시하는지 확인한다.
- [ ] 알림을 허용한 경우 완료 알림, 사운드·진동 설정이 의도대로 동작한다.

### 4. 재부팅 복구

- [ ] 실행 또는 일시정지 상태에서 기기를 재부팅한다.
- [ ] 재부팅 뒤 타이머는 안전하게 `IDLE` 상태로 복구되고, 마지막 설정 시간은 유지된다.

### 5. 무료 스킨 다운로드와 분리 적용

- [ ] 네트워크 연결 상태에서 상점 카탈로그를 연다.
- [ ] 무료 스킨을 다운로드하고 상세 화면과 창고에서 정상 표시되는지 확인한다.
- [ ] 캐릭터 스킨과 타이머 스킨을 각각 선택해 서로 다른 조합으로 위젯에 적용한다.
- [ ] 다운로드 실패 시 재시도 안내가 표시되고 앱이 종료되지 않는다.

## 실패 처리

- 실패한 항목, 재현 순서, 기기/Android 버전, 화면 캡처 또는 로그를 이 문서의 실행 기록과 이슈에 남긴다.
- 기본 타이머·위젯·스킨 다운로드 흐름에서 실패하면 해당 APK는 배포하지 않는다.
